== Description ==

**Overview**

Pop-Up CC Exit Intent plugin brings unlimited pop-ups with **six** easily customizable professional templates. The templates are **responsive** and work with any WordPress theme.

The pop-ups are displayed when a user **intents to leave the page**. They come with **two customizable buttons** with individually assigned URLs.

Live preview: [Default Pop-Up CC Exit Intent template](https://shop.chop-chop.org/demo/pop-up-cc/pop-up-exit/)

**Customization**

Preview the content and the design in our **custom visual editor** to create attractive pop-ups much easier than ever before.

**Features:**

* Unlimited pop-up number
* Beautiful pop-up designs
* 6 elegant and professional responsive pop-up templates
* Retina-friendly
* Quick and easy pop-up setup
* User-friendly interface
* Display control (by user role or by page)
* Two individually customizable buttons

== Installation ==


= Using The WordPress Dashboard =


1. Navigate to the 'Add New' in the plugins dashboard
2. Search for 'chch-pop-up-close'
3. Click 'Install Now'
4. Activate the plugin on the Plugin dashboard
5. Go to PopUp in the left menu to manage the pop-ups


= Uploading in WordPress Dashboard =


1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select 'chch-pop-up-close' from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard
6. Go to PopUp in the left menu to manage the pop-ups


= Using FTP =


1. Download 'chch-pop-up-close'
2. Extract the 'chch-pop-up-close' directory to your computer
3. Upload the 'chch-pop-up-close' directory to the '/wp-content/plugins/' directory
4. Activate the plugin in the Plugin dashboard
5. Go to PopUp in the left menu to manage the pop-ups
